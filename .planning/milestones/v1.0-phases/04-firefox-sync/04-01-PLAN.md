**Exported function: getWorkspaces()**
- Check `isSyncFailed()` first — if true, return `readFromLocal()`
- Try `browser.storage.sync.get(null)` to get all sync data in one call
- If `syncData.wsIndex` exists and is a non-empty array, call `assembleFromSync(syncData)` and return the result
- On any error or missing wsIndex, fall back to `readFromLocal()`
- Log sync failures with `console.warn('[Workspaces] sync.get failed, using local fallback:', e)`

**Exported function: saveWorkspaces(workspaces)**
- Check `isSyncFailed()` first — if true, write to local via `browser.storage.local.set({ workspaces })` and return
- SYNC-03: Call `browser.storage.sync.getBytesInUse(null)` wrapped in try/catch (default to 0 on error)
- Build sync items via `serializeToSyncItems(workspaces)` — this function strips `favIconUrl` from tab objects before chunking (per Pitfall 1 in research)
- Estimate byte size: `Object.entries(items).reduce((sum, [k, v]) => sum + k.length + JSON.stringify(v).length, 0)`
- If `used + estimated > QUOTA_BYTES * QUOTA_THRESHOLD`, log warning and call `activateFallback(workspaces)`, return
- Try `pruneStaleChunks(workspaces)` then `browser.storage.sync.set(items)` in sequence
- Catch errors: if `e.name === 'QuotaExceededError'` or `e.message` includes 'quota' (case-insensitive), call `activateFallback(workspaces)` (SYNC-04). Otherwise re-throw.

**Exported function: migrateIfNeeded()**
- Read `browser.storage.sync.get('wsIndex')` — if `wsIndex` exists, return immediately (idempotent sentinel)
- Read `browser.storage.local.get('workspaces')` — if no workspaces array or empty, return
- Try: call `saveWorkspaces(localData.workspaces)`, then `browser.storage.local.remove('workspaces')`
- Log success: `console.log('[Workspaces] Migrated', count, 'workspaces to sync storage')`
- On error: log warning, do NOT remove local data (keep it intact for fallback)

**Exported function: deleteWorkspaceFromSync(workspaceId)**
- Read `browser.storage.sync.get('ws:' + workspaceId)` to find meta and `tabChunks` count
- Build array of keys to remove: `['ws:' + workspaceId]` plus `['ws:' + workspaceId + ':t:' + i]` for each chunk
- Call `browser.storage.sync.remove(keysToRemove)` wrapped in try/catch (non-fatal on failure, log warning)

**Internal function: serializeToSyncItems(workspaces)**
- Build `items` object with `wsIndex` key (array of workspace IDs in order)
- For each workspace, create metadata key `ws:{id}` containing `{ id, name, color, createdAt, tabChunks: N }`
- Strip `favIconUrl` from each tab before chunking: `tabs.map(t => ({ url: t.url, title: t.title, pinned: t.pinned }))`
- Chunk stripped tabs into arrays of CHUNK_SIZE (25), store as `ws:{id}:t:0`, `ws:{id}:t:1`, etc.
- Use `chunkArray(strippedTabs, CHUNK_SIZE)` helper. If tabs is empty, still produce one chunk with empty array.
- Return the items object.

**Internal function: assembleFromSync(syncData)**
- Read `wsIndex` array from syncData for ordering
- For each ID in wsIndex, read metadata from `syncData['ws:' + id]`
- Skip if metadata missing (orphan ID)
- Reassemble tabs from chunk keys `ws:{id}:t:0` through `ws:{id}:t:{tabChunks-1}`
- Push reconstructed workspace object `{ id, name, color, createdAt, tabs }` (tabs will NOT have favIconUrl — this is expected, browser re-fetches favicons on tab restore)
- Return workspaces array

**Internal function: chunkArray(arr, size)**
- Standard chunk: `for (let i = 0; i < arr.length; i += size) chunks.push(arr.slice(i, i + size))`
- If input is empty, return `[[]]` (always at least one chunk)

**Internal function: pruneStaleChunks(workspaces)**
- Read all sync data via `browser.storage.sync.get(null)`
- For each workspace, compute new chunk count: `Math.ceil(ws.tabs.length / CHUNK_SIZE) || 1`
- Read old chunk count from `syncData['ws:' + ws.id].tabChunks || 1`
- Collect keys `ws:{id}:t:{i}` where `i >= newChunks && i < oldChunks`
- If keys to remove, call `browser.storage.sync.remove(keysToRemove)` in try/catch (non-fatal, log warning)

**Internal function: activateFallback(workspaces)**
- Call `browser.storage.local.set({ [SYNC_FAILED_KEY]: true, workspaces })`

**Internal function: isSyncFailed()**
- Call `browser.storage.local.get({ [SYNC_FAILED_KEY]: false })`
- Return the boolean value

**Internal function: readFromLocal()**
- Call `browser.storage.local.get('workspaces')`
- Return `Array.isArray(result.workspaces) ? result.workspaces : []`
