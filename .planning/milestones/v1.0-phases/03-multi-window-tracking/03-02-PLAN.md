Message contracts (popup -> background):
```javascript
// getState: returns { workspaces, windowMap, currentWindowId, activeWorkspaceId }
{ action: 'getState', windowId: number }

// switchWorkspace: switches this window to target workspace
{ action: 'switchWorkspace', workspaceId: string, windowId: number }

// createWorkspace: creates workspace and switches to it
{ action: 'createWorkspace', name: string, color: string, windowId: number }

// deleteWorkspace: deletes workspace (switches if active)
{ action: 'deleteWorkspace', workspaceId: string, windowId: number }

// updateWorkspace: renames/recolors workspace
{ action: 'updateWorkspace', workspaceId: string, updates: object, windowId: number }

// focusWindow: brings another window to front (WIN-03)
{ action: 'focusWindow', targetWindowId: number }

// assignWorkspace: assigns workspace to unassigned window (D-09)
{ action: 'assignWorkspace', workspaceId: string, windowId: number }

// getColors: unchanged
{ action: 'getColors' }

// forceSave: save current window's tabs
{ action: 'forceSave', windowId: number }
```

getState response shape:
```javascript
{
  workspaces: Array<{ id, name, color, tabs, createdAt }>,
  windowMap: { [windowId: string]: string | null },  // windowId -> workspaceId
  currentWindowId: number,                            // echoed back
  activeWorkspaceId: string | null,                   // this window's active workspace
}
```

  4. Inside the `state.workspaces.forEach` loop, modify workspace item rendering:

     - Add class modifiers to `li`:
