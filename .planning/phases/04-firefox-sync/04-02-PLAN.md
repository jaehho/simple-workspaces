<objective>
Replace all direct browser.storage.local workspace calls in workspaces.js, messaging.js, and index.js with sync.js abstraction functions. Wire migrateIfNeeded() into lifecycle hooks.

Purpose: After this plan, all workspace persistence flows through the sync.js abstraction layer. The extension reads from storage.sync first (synced to Firefox account), falls back to storage.local on errors, and migrates existing local data to sync on update. No code outside sync.js touches browser.storage directly for workspace data.

Output: Three modified files with all 19 storage.local call sites replaced by sync.js imports. Migration wired into onInstalled and onStartup.
</objective>
