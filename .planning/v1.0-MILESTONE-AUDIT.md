### Integration Issue: DATA-03 Fallback Path

`sync.js` `readFromLocal()` returns the raw `workspaces` array from `storage.local` without calling `validateWorkspaceData()`. This path activates when sync quota is exceeded (SYNC-04 fallback). Corrupted local data could reach callers unfiltered.

**Impact:** Low probability (requires sync quota failure + corrupted local storage), but violates the defense-in-depth principle established in Phase 2.

**Fix:** Call `validateWorkspaceData()` inside `getWorkspaces()` on all return paths, or inside `readFromLocal()`.

### Integration Issue: Circular Dependency

`state.js` imports `saveCurrentWorkspace` from `workspaces.js`; `workspaces.js` imports helpers from `state.js`. Works today due to lazy async resolution. Fragile if either module adds synchronous cross-boundary initialization.

**Fix:** Move `throttledSave` into `workspaces.js`, or pass `saveCurrentWorkspace` as parameter from `index.js`.
