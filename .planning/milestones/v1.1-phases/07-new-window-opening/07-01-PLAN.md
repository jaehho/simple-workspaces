Tab creation loop pattern (switchWorkspace lines 147-178):
```javascript
for (let i = 0; i < tabsToCreate.length; i++) {
  const t = tabsToCreate[i]
  const isAbout = !t.url || t.url.startsWith('about:')
  const createProps = {
    windowId,
    active: i === 0,
    pinned: t.pinned || false,
  }
  if (!isAbout) {
    createProps.url = t.url
    if (i > 0) {
      createProps.discarded = true
      createProps.title = t.title || t.url
    }
  }
  try {
    const created = await browser.tabs.create(createProps)
    createdTabIds.push(created.id)
  } catch (err) {
    console.warn('[Workspaces] Tab create fallback for:', t.url, err)
    try {
      delete createProps.discarded
      delete createProps.title
      const created = await browser.tabs.create(createProps)
      createdTabIds.push(created.id)
    } catch (err2) {
      console.error('[Workspaces] Tab create failed entirely:', err2)
    }
  }
}
```

    1. Set `isSwitching: true` via `await setSessionState({ isSwitching: true })` at the top (Pitfall 2 guard — prevents throttledSave from persisting partial state during tab creation in the new window).

    2. Wrap the body in try/catch/finally. The `finally` block always calls `await setSessionState({ isSwitching: false })`.

    3. Inside try:
       a. `const workspaces = await getWorkspaces()` — if empty, `throw new Error('No workspaces found')`
       b. **Exclusive ownership check (D-01, D-02):** `const windowMap = await getWindowMap()`. Iterate `Object.entries(windowMap)`: if any entry has `wsId === targetId`, call `await browser.windows.update(Number(wid), { focused: true })` and return `{ success: true, focusedExisting: true }`. Note: unlike switchWorkspace's check, there is no "callerWindowId" exclusion — ANY window owning the target gets focused, because the caller is opening a *new* window, not switching an existing one.
       c. `const target = workspaces.find(w => w.id === targetId)` — if not found, `throw new Error('Target workspace not found')`
       d. Create new window: `const newWindow = await browser.windows.create({ focused: true })`. Capture the auto-created blank tab: `const blankTabId = newWindow.tabs[0].id` (the `tabs` array is always populated per MDN).
       e. Determine tabs to create: `const tabsToCreate = target.tabs.length > 0 ? target.tabs : [{ url: 'about:newtab', title: 'New Tab', pinned: false }]`
       f. Replicate the tab creation loop from `switchWorkspace()` (lines 147-178) but using `windowId: newWindow.id` instead of `windowId`. The loop creates the first tab as `active: true`, the rest as `discarded: true` with fallback. Collect all created tab IDs in a `createdTabIds` array.
       g. **Rollback on partial failure:** If `createdTabIds.length !== tabsToCreate.length`, close the entire new window via `await browser.windows.remove(newWindow.id)` and return `{ success: false, error: 'New window aborted: not all tabs could be created' }`.
       h. Remove the auto-created blank tab: `await browser.tabs.remove(blankTabId)` (Pitfall 1).
       i. Record MRU: `target.lastUsedAt = Date.now()`
       j. Persist: `await saveWorkspaces(workspaces)`
       k. Record mapping: `await setWindowEntry(newWindow.id, targetId)`
       l. Update badge: `updateBadge(target, newWindow.id)`
       m. Return `{ success: true }`

    4. In the catch block: `console.error('[Workspaces] Open in new window error:', e)` and return `{ success: false, error: e.message }`.
